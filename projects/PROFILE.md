## The auto updating script

I wrote a Python script that reads from my Codeberg pages repo and automatically updates my README.md with the latest logbook & projects entry. Then, I made a GitHub workflow to actually run it. I had to work with GitHub secrets and such so that not only does the script run, but it also commits and pushes to both GitHub (easy) AND Codeberg (harder). I then scheduled it to run in the middle of the night daily and, thanks to the charity GitHub gives out, I now have a script that runs for free every day.
